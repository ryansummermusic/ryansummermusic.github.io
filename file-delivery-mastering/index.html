File Delivery Requirements
(Mastering)

To ensure a smooth workflow and preserve the integrity of your work, please confirm the following details below

Your folder should include:

Reference Mix

A WAV file of the latest mix for referencing purposes

Main Mix (without final limiters or clippers)

- Sample Rate: Same as source
- Bit Depth: 32-bit (Float)
(24-bit is acceptable if no digital clipping occurs at 0dB)

Acapella & Instrumental (optional)

- Sample Rate: Same as source
- Bit Depth: 32-bit (Float)
(24-bit is acceptable if no digital clipping occurs at 0dB)

Important

Before exporting, please ensure that no files exceed 0dBFS.
