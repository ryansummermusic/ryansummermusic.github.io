File Delivery Requirements
(Mixing)

To ensure a smooth workflow and preserve the integrity of your work, please confirm the following details below

Your folder should include:

Reference Mix

A WAV file of the latest mix for referencing purposes

And either of the following two options:

DAW Session
(Pro Tools, Ableton & Logic only)

- Please freeze all tracks, busses & sends with active plugins.
- For Ableton & Logic sessions, please export any non-freezable busses & sends with active plugins.

OR

WAV Multi-Tracks

- A Wet & Dry version (see below for more details)
- Sample Rate: Same as source
- Bit Depth: 32-bit (Float)
(24-bit is acceptable if no digital clipping occurs at 0dB)

Important

Before exporting, please ensure that no files exceed 0dBFS.

Guidelines for exporting WAV Multi-Tracks

Wav files should be exported as follows, although please reach out for confirmation as some versions may not be required. 

Vocals

✅ A Processed/Wet Stems folder:

(Leaving on all processing, effects, volume balance & automation.
Additionally, exporting any grouped tracks, buses & sends)

✅ An Unprocessed/Dry Stems folder:

(Leaving only pitch/timing correction - or any other processing you wish to keep intact).

Instrumental

✅ A Processed/Wet Stems folder:

(Maintaining all processing, effects, volume balance & automation as heard in the rough mix).

✅ An Unprocessed/Dry Stems folder (Optional):

(Feel free to include the Kick, Bass, or any other instruments if unsure of any processing).
